Verifier Privacy

Local-First Verification

This verifier is designed to process QR verification in the browser. No account login is required for verification workflows.

  • QR scan and signature checks run in client-side JavaScript.
  • Credential details are read only from signed QR payloads when available.
  • No persistent student profile is created by this static verifier.
  • Result state is stored in session storage for page navigation continuity.
